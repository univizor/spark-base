# univizor / spark-base

[![Build Status](https://travis-ci.org/univizor/spark-base.svg?branch=master)](https://travis-ci.org/univizor/spark-base)

Build with

```bash
docker build -t univizor/spark-base .
```.

Required to build [spark-docker][spark-docker] container.
